## 📂 3. 目录结构与代码提交规范
大家在自己的分支里写好代码后，请将代码文件（`.py` 或 `.ipynb`等）提交到对应的文件夹内：
* `/data` : 存放数据预处理、清洗、增强等相关脚本
* `/model` : 存放模型结构定义及训练（Training）代码
* `/analysis` : 存放错误样本分析和指标分析的 Jupyter Notebook 或可视化图表
* `/frontend` : 存放前端 UI 界面代码
* `/src` : 存放负责调用模型、处理推断（Inference）并与前端连接的核心代码
